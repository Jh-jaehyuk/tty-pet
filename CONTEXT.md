# tty-pet Context

`tty-pet` is a playful terminal companion for a developer's current project. This glossary keeps the product language precise so the pet stays a toy-like companion rather than becoming an assistant or productivity monitor.

## Language

**Pet Interaction**:
An explicit event directed at the pet, such as poking, treating, calling, napping, or marking a test result.
_Avoid_: Agent interaction, command result

**Agent Presentation**:
The way an agent client describes a pet interaction, status, or reaction back to the user.
_Avoid_: Pet interaction, status output

**Pet Interpreter**:
The agent role for presenting pet state with a small amount of personality while preserving the actual state.
_Avoid_: Development coach, assistant persona

**Reaction**:
A short-lived pet response caused by a recent pet interaction or project state change.
_Avoid_: Workflow action, automation
